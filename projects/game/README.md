# Wake Up Prithvi

## setting

The setting is in Prithvi's home in the morning.

## Map

![map](projects/game/GameImages/FlowChart.png)

## Story

The player must help a student named Prithvi wake up and get ready for school before time runs out.

## Variables

- Time - creates sense of urgency and actually allows you to lose
- EnergyLevel - affects how quickly Prithvi moves between rooms and whehther or not he falls back asleep
